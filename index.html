Reformatting is a natural process which generates the ordinary temporal collection of self-development as an entirely new selfhood found in the moment of performance. Only the self which is in utterance is its actual format. Such, the progression of time allows for reformatting in infinite repetition.



This site:

  1. Compiles and reformats all keystrokes of three creative workers working discursively on The Actual School's seminar, A Flexible Anarchy from 04/02/15 to 04/16/15
  2. Compresses this two week time frame into a 24 hour cycle
  3. Generates each keystroke at the minute it was originally typed


By displaying the processes behind this seminar, there is an aim to reformat by removing the “self” behind each typist/worker, and to reveal the meta- and backend aspects of their creative functioning. The soft-constructed self — the one which is upon the articulation of being — is unable to reveal its previous format since the three workers have been compiled into a single text-entity, only presented partially within a user's session. As a result, any displayed “self” begins only at the minute the user interacts with the page. The flexibility exists within different formats of self the user can witness between sessions: a reformatting.
